Project Overview

This project involved setting up a DHCP server on a Cisco router to automatically assign IP addresses to network devices. Instead of manually configuring each device with an IP address, subnet mask, and gateway, DHCP handles all of this automatically.

I created an IP address pool of 50 addresses, configured lease times, set up address exclusions for servers and printers that need static IPs, and tested various scenarios including IP exhaustion and lease renewal behavior.

The Problem

New devices joining the network had no IP addresses. Manually configuring each device would take forever and could cause IP conflicts if two devices got the same address.

The Solution

Set up a DHCP server that automatically assigns unique IPs from a pool. Each device gets an IP, gateway, and DNS info without any manual configuration required.

The DORA Process

DHCP uses a four-step process called DORA to assign IP addresses to clients:

D

Discover

Client broadcasts looking for a DHCP server

O

Offer

Server offers an available IP address

R

Request

Client requests the offered IP

A

Acknowledge

Server confirms the assignment

What I Did

1

Created Network Topology

Set up 1 router, 1 switch, and 5 PCs in GNS3 to simulate a small office network environment.

2

Configured DHCP Pool

Created pool "OFFICE" with IP range 192.168.1.10 to 192.168.1.60 and set the default gateway as .1

3

Set Address Exclusions

Excluded addresses .1 to .9 for servers and printers that require static IP addresses.

4

Configured DNS Options

Added DNS server addresses (8.8.8.8 and 8.8.4.4) to the DHCP options for client resolution.

5

Tested Client Acquisition

Set each PC to "obtain IP automatically" and verified they received unique addresses from the pool.

6

Tested Edge Cases

Simulated lease expiry, IP exhaustion, and DHCP server failure scenarios to understand behavior.

Configuration Commands

DHCP Server Configuration
Router(config)# ip dhcp pool OFFICE
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8 8.8.4.4
Router(dhcp-config)# lease 7
Router(dhcp-config)# exit
Address Exclusions
Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.9

! Reserve addresses for:
! .1 = Default Gateway
! .2-.5 = Servers
! .6-.9 = Network Printers
